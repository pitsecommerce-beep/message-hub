# Legacy - Codigo Anterior (No Activo)

Esta carpeta contiene la version anterior de MessageHub, construida con **HTML + CSS + JavaScript vanilla**.

## Archivos

| Archivo      | Descripcion                                  |
|-------------|----------------------------------------------|
| `app.js`    | Aplicacion completa en vanilla JS (~5,100 lineas) |
| `index.html`| Estructura HTML con todos los modales y vistas |
| `styles.css`| Estilos CSS completos de la app anterior      |

## Estado

- **No se despliega** - La app activa esta en `apps/web/` (React + TypeScript + Vite)
- **No se usa** - Ninguna parte del codigo activo depende de estos archivos
- **Solo referencia** - Se conserva como respaldo y referencia historica

## App Activa

La aplicacion actual se encuentra en:
- **Frontend**: `apps/web/` (React 19, TypeScript, Vite, Tailwind CSS)
- **Backend**: `functions/` (Firebase Cloud Functions)
- **Deploy**: GitHub Pages via GitHub Actions (`apps/web/dist/`)
